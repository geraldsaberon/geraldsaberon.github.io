Solving an 8-tile sliding puzzle game using graph traversal/search algorithms, [Breadth-first Search (BFS)](https://en.wikipedia.org/wiki/Breadth-first_search) and [A* Search](https://en.wikipedia.org/wiki/A*_search_algorithm)

Uses [Manhattan Distance](https://en.wikipedia.org/wiki/Taxicab_geometry) as the heuristic function for A* search.
